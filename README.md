# 💸 Expense Tracker – Personal Budget Manager

A simple and effective web app to record daily income and expenses, calculate balance, and manage your budget — built using *HTML, CSS, and JavaScript*.

---

## 🚀 Features
- ➕ Add new income or expense entries  
- 💰 Auto-calculate total balance, income, and expenses  
- 💾 Saves data in *LocalStorage* (persists even after refresh)  
- 🧹 Delete transactions easily  
- 📱 Fully responsive and clean layout  

---

## 🧠 Tech Stack
- *HTML5* – Structure  
- *CSS3* – Styling and layout  
- *JavaScript (ES6)* – Logic and interactivity  

---

## ⚙ How It Works
1. Enter a description and amount (+ for income, - for expense)  
2. Click “Add Transaction”  
3. The list and balance update instantly  
4. Data stays saved using LocalStorage
